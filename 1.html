HTML

Hypertext Markup Language(HTML) is the standard markup language for creating web pages and web applications. Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML despcribes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the builing blocks of HTML pages. With HTML contructs, images and ohter objects, such as interactive forms, may be embedded into the rendered page.
